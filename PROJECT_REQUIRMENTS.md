Knapsack Problem: Develop a program that solves the Knapsack Problem using different algorithms (e.g.
brute force, greedy algorithm, dynamic programming). The project can be divided into algorithm
implementation, frontend development, and performance evaluation. A team of 2-3 members would be ideal.

Deliverables:
The following should be uploaded on the course Moodle site before the discussion. Else, you will get zero for
all elements related to documentation.
• Executable program + The source code
• Presentation (Slides) + Report that describes the work
• The team’s work typically should include illustrations of the implemented algorithms and their analysis,
both theoretical and empirical if possible.
